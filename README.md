# Launch UI

This is Frontned project which is used to distribute the components that are currently avaiable on the website. By components i mean everything you see on the website, so instead of creating again and again.. directly copy and paste this beautiful ui.

## Deployed Link

View it live here:-> https://launch-ui-rust.vercel.app/

## Tech Stack

Next.js + Tailwind.css + framer motion(Now motion)
Vercel for deployment
